DOMQL

Tired of using document.querySelector() with CSS selectors? Ever wanted to use SQL-like syntax to query DOM elements?
Yeah, me neither.

To use the library, simply use one the following methods to import the library.
To compile a query, use the domql.Domql constructor
To run a compiled query, use the .execute() method.

ES6: 


        import Domql from './domql.min.js';
        var cursor = new Domql("SELECT * FROM html WHERE id='foo'");
        var result = cursor.execute();
From script: 

        <script src="https://cdn.jsdelivr.net/gh/yuxuan-ji/domql@master/lib/domql.min.js"></script>

        <script>
            var cursor = new domql.Domql("SELECT * FROM html WHERE id='foo'");
            var result = cursor.execute();
        </script>
